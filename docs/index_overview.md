# Общий обзор индексов ZERNO

Индексы ZERNO отражают совокупную доходность и оценку компаний Мосбиржи, с учётом:

- актуальной рыночной цены
- LTM/NTM дивидендной доходности
- ERP (премии за риск)
- ключевой ставки ЦБ
- прогноза доходности сектора

Каждый индекс — агрегатор данных по всем компаниям в выборке.  
Вес компании зависит от ликвидности и стабильности дивидендных выплат.
