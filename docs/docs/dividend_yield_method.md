# Методика расчёта дивидендной доходности LTM и NTM

## LTM (Last Twelve Months)
LTM отражает фактические дивиденды, выплаченные за последние 12 месяцев:

LTM Yield = (Σ дивидендов за 12 месяцев) / текущая цена × 100%


## NTM (Next Twelve Months)

Оценка NTM формируется по модели следующего года:

- последняя дивидендная политика компании
- историческая стабильность
- EPS / FCF
- макропрогноз сектора

NTM Yield = прогнозируемый дивиденд / текущая цена × 100%
